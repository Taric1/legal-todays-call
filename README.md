# legal-todays-call

Privacy policy for the **Today's Call** app, published with [GitHub Pages](https://pages.github.com/).

## Publish

1. Push this repository to GitHub (this folder is the repo root).
2. On GitHub: **Settings → Pages → Build and deployment → Source**: *Deploy from a branch*.
3. Choose branch **main** and folder **/docs**, then save.

The public URL will be:

**https://taric1.github.io/legal-todays-call/**

Use that URL in App Store Connect, Google Play, and in the app (`AppConfig.privacyPolicyUrl`).

## Before launch

- Replace the placeholder contact email in `docs/index.html`.
